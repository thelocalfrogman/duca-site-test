## Workshop Overview

This hands-on workshop will teach you essential network security skills used by professionals in the industry.

### Topics Covered

1. **Packet Analysis with Wireshark**
   - Capturing network traffic
   - Analyzing protocols
   - Identifying suspicious activity

2. **Firewall Configuration**
   - Understanding firewall rules
   - Common firewall architectures
   - Best practices

3. **Intrusion Detection Systems**
   - How IDS/IPS works
   - Setting up Snort
   - Creating custom rules

## Prerequisites

- Basic understanding of networking concepts (TCP/IP, ports, protocols)
- Laptop with Wireshark installed

## What You'll Take Away

- Practical skills in network analysis
- Understanding of defensive security measures
- Resources for further learning
