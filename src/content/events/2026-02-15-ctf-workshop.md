## About This Workshop

Join us for an introductory workshop on web exploitation techniques commonly found in CTF competitions. This session is designed for beginners who want to learn the fundamentals of web security testing.

### What You'll Learn

- Understanding common web vulnerabilities (SQL injection, XSS, CSRF)
- Using browser developer tools for security testing
- Basic reconnaissance techniques
- Hands-on practice with beginner-level challenges

### Requirements

- Laptop with a modern web browser
- Basic understanding of HTML and HTTP
- Curiosity and willingness to learn!

### Schedule

- 2:00 PM - Introduction and setup
- 2:30 PM - Web vulnerability overview
- 3:30 PM - Hands-on challenges
- 5:00 PM - Wrap-up and Q&A

Snacks and drinks will be provided. Don't forget to join our Discord for updates and to connect with other participants!
